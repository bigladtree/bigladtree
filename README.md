# About Me!!
Hello, I'm Chris!! I'm a Computer Science student at Central Michigan University with a passion for front end/game development. I really enjoy working with C# and Java, but am working on honing my skills in all common programming languages. Currently I'm in my second year at CMU, so I am pretty good at Java and am actively working on bettering my C and C# skills. I love learning new tech and making my own projects! 

## Big Projects I've Worked On
QuickStay System - A Group Project developed with 4 of my classmates (Githubs: EthanPhillip04, beccabanks, leahvan-cmu, AveryLong-H). I created the CSS, connected every screen together, made the home screen (after the log in), and made the table system used in the View All Properties and Filter Properties screens, as well as debugged in general.

Iron Ascent - A Group Unity Project developed with 3 classmates (Githubs: StallmanDarl, plunket81). I was in charge of 2 of the levels, all the character animation, and did general bug fixing (namely on character movement). I also coded the rangers AI and implemented the enemy AI into the character models, all with their own settings and animations. 

CatScout - a project for the hackathon #hackthekitty! This was by me and @leahvan-cmu on github. Catscout is an adoption site that scrapes shelters for cat profiles to display to the users. I was personally in charge of all the front end things, so I formatted all the html files, coded all the css, and generally was in charge of the UI/UX. Some staple things I did was get Leahs web scraper to appear on the website and make it so you were able to save the cats for later.
